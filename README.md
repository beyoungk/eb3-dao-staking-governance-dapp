# EB3 DAO STAKING GOVERNANCE DAPP

## Purpose
This project is my main Web3 career portfolio project where I am building a DAO staking and governance DApp to practice the core skills needed for Web3 frontend, smart contract, and product roles.
The purpose is to understand the full product lifecycle of a Web3 application by planning the product, designing token and governance logic, building the frontend, writing smart contracts, testing the system, and documenting risks.
